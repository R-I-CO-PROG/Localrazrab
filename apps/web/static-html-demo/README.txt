ULT Concept AI — статическая демо-версия фронтенда
====================================================

Как открыть:
1. Распакуйте архив в любую папку
2. Откройте файл index.html двойным щелчком (Chrome, Edge, Firefox)
3. Сервер и интернет не нужны — всё работает локально

Что внутри:
- Все основные экраны интерфейса (дашборд, генерация, проекты, визуализации и др.)
- Демо-данные и заглушки вместо реальных AI-фото и API
- Кнопки генерации показывают демо-уведомление (без бэкенда)

Страницы:
  index.html              — главная
  generate.html           — форма генерации
  concepts.html           — мои проекты
  concepts-results.html   — 5 концепций
  concept-detail.html     — карточка набора
  visualizations.html     — галерея визуализаций
  proposals.html, brandbooks.html, templates.html, favorites.html, settings.html

Примечание: это UI-preview для демонстрации дизайна, не рабочая версия с API.
